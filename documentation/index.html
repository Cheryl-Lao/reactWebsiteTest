Fleja - React Documentation

Introduction

Fleja is a creative modern multi-concept React template. This is built with React.js, Next.js, Sass, etc. The template contains demo for 8+ business niches including Digital Agency, Startups, Creative Agency, Freelancer or Blog and more niches will be added soon. The template also has multiple pages for different website needs.

Setup Development Environment

To kickstart the development of the web app with Fleja, at first need to setup the react development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

Preparation

You'll need to install Node.js >=v8.11+ (Recommended Version) (NPM comes along with it) and serve (a NPM package)

Use command line tool i.e. powershell or terminal and navigate to the project(Fleja) root. This is powershell in windows and terminal on mac/linux.

Let's have a look on the setup process:

Use terminal and navigate to the project root.

Then run : npm install

Then run : npm run dev

It'll return something like this in the console:

> project@1.0.0 dev drive:\project

> next

[ wait ] starting the development server ...

[ info ] waiting on http://localhost:3000 ...

[ ready ] compiled successfully

[ info ] ready on http://localhost:3000

Now, in the browser go to localhost:3000

For Static Build

Run : npm run build

Then Run : npm run export

Then Run : npm install -g serve

Then Run : cd out

Then Run : serve -p 8080

Now you can access your static app in http://localhost:8080. Everything should be working as usual. For more info please visit the Official Doc.

Fonts

By default, the template loads Open+Sans and Dosis font from Google Web Font Services, you can change the font with the one that suits you best.

You will find the font code in the "style.scss or style.css" file: /assets/css/style.scss or /assets/css/style.css


@import url('https://fonts.googleapis.com/css?family=Open+Sans:300,300i,400,400i,600,600i,700,700i&display=swap');
@import url('https://fonts.googleapis.com/css?family=Dosis:200,300,400,500,600,700,800&display=swap');
